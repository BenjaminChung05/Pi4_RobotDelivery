Here lies the main code to be used by anyone. The main function of this code would be to follow Red, Yellow and Black Line
while being able to detect symbols and perform said action. 

List of action: 
Hazard/Button symbol detection meant robot car stop moving for a predetermined time.
Arrow symbols meant for robot car to know which direction should it travel when approaching a junction. Must be based on arrowhead and not colour
Fingerprint/QR will be used to identified and shows in user terminal on it has detected a QR symbol or Fingerprint symbol

Another branch in this repo would be the version history created before this Final File can be used. Different method have been tested out and this has proven to be the most successful ones.
