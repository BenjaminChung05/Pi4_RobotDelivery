# Read Me for Desciption
Here lies the main code to be used by anyone. The main function of this code would be to follow Red, Yellow and Black Line
while being able to detect symbols and perform the said action as below: 

### List of action: 
1. Hazard/Button symbol detection meant robot car stop moving for a predetermined time.
2. Arrow symbols meant for robot car to know which direction should it travel when approaching a junction. Must be based on arrowhead and not colour
3. Fingerprint/QR will be used to identified and shows in user terminal on it has detected a QR symbol or Fingerprint symbol
